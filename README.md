# Tic Tac Toe — Solo or Duo!

Simple HTML/CSS/JS implementation of Tic Tac Toe supporting Two Player and Single Player (vs basic AI).

Features
- Two-player mode
- Single-player vs computer with smart move priorities (win -> block -> center -> corners -> sides)
- Scoreboard, restart round, and reset all
- Responsive board and simple animations

Files
- `index.html` — main UI
- `style.css` — layout, responsive styles, animations
- `script.js` — game logic and AI
